# 이벤트 안내
홍보숲 서버 내 `#📢이벤트` 채널에서 이벤트를 진행합니다!
이벤트는 **자동 홍보권**을 걸고 랜덤 추첨방식으로 진행됩니다!

# 자동홍보권은 무엇인가요?
자동홍보권은 홍보숲 서버 내 `#📡자유홍보` 채널에 4시간에 1번씩 자동으로 홍보를 해드립니다. (기간 : 1달)
이제 자동으로 홍보를 하실 수 있어요!

# 주의사항
+ 당첨된 유저가 서버를 나가신다면 홍보는 삭제됩니다.
+ 관리자도 참여가 가능합니다.
